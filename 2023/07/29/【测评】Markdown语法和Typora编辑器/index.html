【测评】Markdown语法和Typora编辑器

发表于2023-07-29|更新于2024-02-16
|阅读量:

你写博客用的是什么语法呢？这是我一直想和大家讨论的问题，有的人用的可能是html，有的人可能直接撰写，像Word文档一样，还有些人用富文本编辑器，但是我相信，大部分人用的还是Markdown语法，我之前在博族论坛上发了个贴，题目是《大家都用什么md编辑器》（这里md就只Markdown语法），各位大佬反正是记事本，备忘录，vscode idea什么都能用，也有些人可能也会vditor这种在线编辑器。而今天我在讲解什么是Markdown语法的同时，也顺道测评一下Typora。这篇文章从7月初开始撰写，但因为众多原因导致大半个月后才发布，这里，我表示非常的抱歉！

介绍Markdown

在看这篇文章之前，可能很多人还不太了解Markdown，话不多说，直接上百度百科

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。 它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者HTML）文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。
由于 Markdown 的轻量化、易读易写特性，并且对于图片，图表、数学式都有支持，许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表消息。 如 GitHub、Reddit、Diaspora、Stack Exchange、OpenStreetMap 、SourceForge、简书等，甚至还能被使用来撰写电子书。

以上内容摘选自百度百科

看完这个，你应该大体的了解了Markdown语法（下面简称MD语法），现在，大多数博客都采用MD语法，有人说，我用Word编辑是不是可以达到同样的效果，其不然，Word的用途只是用于展现出一些可以打印出来的东西，而MD语法，主要是在电脑上显示，它可以让你的文章里插入音乐、视频、图片甚至外链，更加方便了博主的撰写以及访客的阅读！

介绍一下Typora

想要写好Markdown语法，你首先需要一个MD编辑器，有人说，我用记事本写，在改一个后缀，也是一个MD文件，并不需要额外去下载甚至购买付费的MD编辑器！

首先，假如你是大佬，精通MD语法，你当然可以去用记事本，毕竟，记事本可是万能的！但，假如你是个小白，并不熟悉MD语法，那么，你极其需要一个好用的MD编辑器，很早之前，我用hexo博客，那时候也在为MD编辑器发愁，便在博族论坛 上发帖咨询一下各位大佬，得到的回复也是各不相同。

而我最后选择的是Typora 编辑器，这是一款收费的编辑器，售价88/3个授权，我是通过从杜老师那里花费30大洋买了一个授权，体验效果还是非常不错，感觉还是物有所值，功能齐全，目前我写文章都是在本地先用Typora写好后，在传到Typecho上，感觉比在线编辑器好用太多了！

其他编辑器

当然，还有很多其他编辑器供你选择，如果不想安装，也可以选择在线编辑器，这完全看你的个人喜好
